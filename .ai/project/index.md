## Customization Guide

### Tech Stack

Update `tech-stack.md` with your project's:
- Framework choices
- Database technology
- Styling solution
- Testing frameworks
- Package manager

### Discovered Patterns

This file is a living log of patterns discovered during development:
- AI agents add patterns they find
- You review and promote important ones
- Outdated patterns get archived

---

## Template Values

Throughout the `.ai/` documentation, you'll see placeholders like:
- `{{PROJECT_NAME}}`
- `{{FRONTEND_FRAMEWORK}}`
- `{{DATABASE}}`

Replace these with your actual choices in `tech-stack.md`.
